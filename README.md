# dotfiles
 
## Hyprland

### hypr/hw/hypr.conf

Imported at the start of the hyprland configuration.
Set things like workspaces, monitor layout, etc.

### hypr/hw/lock.conf

set screen with $screen = xyz
set wallpaper with $wallpaper = xyz

### hypr/hyprpaper.conf

regularly set the wallpaper with it
